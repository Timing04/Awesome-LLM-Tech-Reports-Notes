# Evaluation and Benchmark Reading

## Scope

这个专题关注如何阅读模型报告里的 benchmark，不只看分数，还看任务构成、污染风险、可比性和统计意义。

## Key Questions

- benchmark 分数为什么经常不能直接横比
- 通用 benchmark 和真实任务表现的关系是什么
- 应该如何读懂模型报告里的表格和胜负结论
