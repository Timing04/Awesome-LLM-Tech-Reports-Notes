# Post-training

## Scope

这个专题关注预训练之后的能力塑形，包括 SFT、偏好优化、强化学习、拒答和安全行为对齐。

## Key Questions

- SFT 能解决什么，不能解决什么
- PPO、DPO、GRPO 等路线分别适合什么场景
- 为什么后训练往往决定最终产品体验
